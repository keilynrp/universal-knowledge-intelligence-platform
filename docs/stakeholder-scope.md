# Business Stakeholder Scope

## Capability-to-Stakeholder Mapping

### Executive / Leadership

**Goal:** Strategic decision-making from research portfolio intelligence.

| Capability | How It Helps |
|------------|--------------|
| Executive Dashboard | KPIs, enrichment coverage, timeline trends at a glance |
| Decision Readout | Structured signals, confidence levels, recommendations |
| Audience Presets (leadership) | Framing emphasizes strategic implications and ROI |
| Quality Score | Single metric for data readiness and reliability |
| PDF/Excel Reports | Shareable briefs for board meetings and reviews |

### Data Steward / Research Office

**Goal:** Ensure data quality, govern canonical identifiers, manage enrichment.

| Capability | How It Helps |
|------------|--------------|
| Source Profiling | Understand incoming data before mapping |
| Mapping Suggestions | AI-assisted field mapping with review workflow |
| Authority Resolution | Deduplicate entities via registries (ORCID, ROR, Wikidata) |
| Layer Boundaries | Provenance preserved — rollback without data loss |
| Harmonization Pipeline | Multi-step data cleaning with undo/redo |
| GenAI Governance | AI outputs require evidence and review before canonical |
| Audit Log | Full trail of who changed what and when |

### Analyst / Investigator

**Goal:** Discover patterns, explore relationships, build evidence.

| Capability | How It Helps |
|------------|--------------|
| Topic Modeling | Concept co-occurrence (PMI), cluster detection |
| Field Correlation | Cramer's V analysis across entity attributes |
| OLAP Cube Explorer | Cross-tabulation with filters and drill-down |
| RAG Search | AI-assisted evidence retrieval grounded in platform data |
| Evidence Traceability | Panels showing provenance for every claim |
| JSON-LD Export | Linked-data output for integration with external tools |
| Graph Export | Relationship visualization and network analysis |

### Innovation / Technology Transfer

**Goal:** Identify commercialization opportunities and partnerships.

| Capability | How It Helps |
|------------|--------------|
| Audience Presets (innovation_transfer) | Framing emphasizes IP potential and partnerships |
| Entity Linking | Find connections between researchers, institutions, concepts |
| Geographic Relationships | Map research activity by location |
| Institution Reconciliation | Normalize affiliations for partnership analysis |
| External Attention | Track citations, altmetrics, and impact signals |

### Evaluator / Compliance

**Goal:** Assess research output quality and compliance.

| Capability | How It Helps |
|------------|--------------|
| Audience Presets (evaluator) | Framing emphasizes metrics, benchmarks, standards |
| Quality Metrics | Field completeness, enrichment coverage, authority resolution rates |
| Evidence Grading | Relevance, quality, and recency scoring per evidence item |
| Citation Grounding | Map claims to supporting evidence references |
| Provenance Badges | Visual treatment distinguishing source/enriched/canonical/authority |

## Audience Preset Summary

| Preset | Primary User | Emphasis |
|--------|-------------|----------|
| `leadership` | C-suite, VP Research | Strategic signals, ROI, portfolio health |
| `research_office` | Data stewards, librarians | Data quality, governance, compliance |
| `investigator` | Researchers, analysts | Discovery, evidence, exploration |
| `innovation_transfer` | TTO officers | IP, partnerships, commercialization |
| `evaluator` | Review panels, auditors | Metrics, benchmarks, standards |
